# NGDG2 (NoGaDa Game 2)

## Summary
   
기존 노가다게임1은 단순히 두 개의 버튼을 번갈아 누르면서 레벨업을 했지만   
노가다게임2는 던전에 입장해서 몬스터들을 잡고 장비를 파밍해서 스펙업을 할 수 있게 설계해서 좀 더 RPG의 재미를 느낄 수 있습니다.
